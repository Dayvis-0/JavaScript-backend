Tipos de datos primitivos

¿Que es un tipo de dato?
Un tipo de dato describe a conjunto de valores y las operaciones posibles sobre esos valores
¿Que es un tipo de dato primitivo?
Un tipo de dato primitivo es una de las formas mas simples de datos

  • Number

    • El tipo de datos number engloba a entero es un numero entero sin un punto decimal.Es de 64 bits

    flotante es un numero con punto decimal

    +infinity es un valor numerico que representa infinito

    -infinity es un valor numerico que representa menos infinito

    y NaN es un valor que representa Not-a-Number

  • BigInt

    • Representan valores numéricos que son muy grandes para ser representados por el primitivo number
    Un bigint es un primitivo, creado al anexar n al final de un literal entero o llamando a una funcion BigInt()

  • String

    • Se utilizan para representar datos textuales. Es un conjunto de "elementos" de valores enteros sin signo de 16 bits

    Acceder
    Se puede acceder a un caracter indiviual en una cadena.
    La primera es con el metodo charAt() y la otra es tratarlo como un arreglo, donde los caracteres indiviuales corresponden a un indice numerico

    Comparar
    Se puede comparar con operades de menor que o mayor que

  • Boolean

    • Es un dato logico que solo puede tener los valores true o false

    Objeto boolean
    El valor pasado como primer parametro se convierte en booleano, si es necesario.

    Si el primer valor se omite o es 0, -0, null, false, NaN, undefined, o la cadena vacia (""), el objeto tiene nu valor de false:

    Todo lo demas valores, incluido cualquier objeto, un arreglo vacio([]) o la cadena "false", crea un objeto con un valor incial de true.
    Cualquier objeto cuyo valor no sea undefined o null, incluido Boolean cuyo valor es false,
    se evalua como true cuando se pasas a una declaracion condicional

    Este comportamiento no se aplica a los booleans primitivos

    No utilices un objeto boolean para convertir un valor no booleano en un valor no booleano. Para realizar esta tarea,
    en su lugar, usa boolean como funcion, o un operador NOT doble, ya que el objeto boolean
    devuelve un objeto, no un valor boolean primitivo y ese objeto siempre es Truty (se comporta como true en condicionales)
