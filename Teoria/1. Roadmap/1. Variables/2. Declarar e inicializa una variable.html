Declarar e inicializa una variable

No coonfundir una variable que existe pero no tiene un valor definido, con una variable que no existe en absoluto son cosas muy diferentes Usando la analogia de una caja se puede decir: No existir significaria que no hay caja (variable) para guardar un valor Ningun valor definido singnificaria que hay una caja, pero no tiene ningun valor dentro de ella.
